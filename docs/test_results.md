## How to fill this table

1. Upload the document and run it through the full pipeline.
2. Compare the extracted result (before and after review) against the
   source document.
3. Mark each column Yes/No/Partial.
4. Record the OCR quality score shown on the Processing Status page.
5. Note whether NER added any useful person/company records the
   rule-based extraction missed.
6. Use the Notes column for anything unusual (e.g. scan quality, unusual
   layout, name-splitting edge cases).
