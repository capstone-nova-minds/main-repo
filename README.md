# Arabic Court Document Extraction

## Project Overview

An AI-powered application for processing Arabic Jordanian court attachment orders and extracting structured information such as customer names and National IDs.

## Current Status

This project is currently under active development. OCR, extraction accuracy, validation, and user-interface components are still being improved.

## Current Features

- PDF, JPG, and PNG upload
- Single-page and multi-page document processing
- Arabic OCR processing
- Initial structured field extraction
- Human review and validation
- JSON and Excel export

## Project Scope

### Included

- Document upload
- OCR
- AI-based document understanding
- Structured information extraction
- Human review
- JSON and Excel export

### Excluded

- Customer database lookup
- Wallet freeze APIs
- External system integration
- Workflow automation
