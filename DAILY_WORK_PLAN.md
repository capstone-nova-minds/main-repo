## 1. Sprint Goals

By the end of the sprint, our team aims to deliver:

- A working AI capstone project.
- A clean GitHub repository with documentation.
- A tested pipeline with clear evaluation results.
- A demo video or live demo.
- A presentation deck for Demo Day.
- A public portfolio artifact.
